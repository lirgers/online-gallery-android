# Gallery

Simple photo gallery that gets images from unsplash.com.

Steps to run:
0. Install and run android emulator (https://developer.android.com/studio/run/managing-avds)
1. npm install
2. react-native eject
3. https://github.com/luggit/react-native-config && https://github.com/luggit/react-native-config/issues/299
4. create .env file in root directory and specify your own client_id for unsplash.com e.g.:
> CLIENT_ID=blablabla
5. react-native run-android
